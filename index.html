Philosophy

Ethics

Even in something as simple as writing an article, it's important to consider ethics. User consent is a very important consideration. For example:

  • Do not track users' behaviour
    • If tracking behaviour is necessary, inform users first and give them an opportunity to opt out
  • If potentially triggering topics will be discussed, warn users in time to give them the chance to prepare themselves or to disengage first

For code that may be reused, consider using an ethics-focused licence such as the Hippocratic Licence instead of a typical "Open Source" licence, to try to prevent it being used in harmful ways.

Developers have a moral duty to try to prevent their work being used for harm. Educate yourself on how to push back when asked to do work that you consider could be harmful.

One potentially useful example of this is the JSON licence, which contains a clause saying "The Software shall be used for Good, not Evil".

General accessibility principles

The Web Content Accessibility Guidelines (WCAG) is the standard for web accessibility.

General principles that are useful to remember when building a website include that it should…

  • work with keyboard-only navigation
  • be usable with a screen reader
  • be usable by colourblind people

General usability principles

A website should:

  • be usable on any device
  • be able to be printed
  • be usable without JavaScript, where possible
