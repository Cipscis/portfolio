Status

This can be used to show global success or error messages, relying on the presence of an element on the page that should be using position: fixed;. Default styles can be found in status.scss.

These default styles use a combination of CSS animations with animation-fill-mode: forwards; to show and hide the status message automatically.

Status message can be invoked either via the success or error methods on the statusMessage function, or by publishing the /status/success or /status/error events.
